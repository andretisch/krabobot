Диалоги

    krabobot

    Локально: тот же origin, что и krabobot serve.

    Подключение

    Страница и API с одного хоста (без CORS): запросы идут на /v1/….

    Модель
    —
    Сервер
    —
    Эндпоинты
    GET /v1/models, POST /v1/chat/completions, GET /v1/web/sessions …

    Сессия в браузере

    Идентификатор в localStorage связывает вкладку с файлом истории на сервере (api:<uuid>).

    Вложения

    Загрузки из чата пишутся в рабочий каталог агента: uploads/web/<session_id>/ (UTF-8 для текста, бинарники как есть).
